# ApniParampara 🪷

A beautiful web platform celebrating the rich cultural heritage of India, showcasing the diverse traditions, art, food, and festivals across different Indian states.

## Overview

ApniParampara is an interactive web application dedicated to preserving and promoting Indian cultural traditions. The platform provides an immersive experience with curated content about various Indian cultures, their unique characteristics, artistic expressions, traditional cuisines, and vibrant festivals.

## Features

- **🎭 Cultural Heritage**: Explore the unique cultural traditions of different Indian states
  - Gujarat Culture
  - Maharashtra Culture
  - Rajasthan Culture

- **🎨 Art & Craftsmanship**: Discover traditional Indian art forms and handicrafts

- **🍛 Food & Cuisine**: Learn about authentic traditional recipes and culinary traditions

- **🎉 Festivals**: Experience the vibrancy of Indian festivals and celebrations

- **📱 Responsive Design**: Fully responsive interface that works on desktop, tablet, and mobile devices

- **🎥 Multimedia Content**: Videos and images showcasing cultural elements

- **🔄 Interactive Sliders**: Smooth carousel/slider functionality for browsing content

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/May-Jo/apni_parampara.git
cd apni_parampara
```

2. Open the application:
   - Simply open `cp1.html` in your web browser
   - Or use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```
   - Then navigate to `http://localhost:8000`

## Usage
